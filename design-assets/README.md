# Design Assets - Financial Copilot AI

Esta carpeta contiene todos los recursos de diseño y branding del proyecto.

## Estructura

```
design-assets/
├── branding/           # Logo, eslogan, guías de marca
├── mockups/            # Diseños de UI/UX (opcional)
├── icons/              # Iconos personalizados (opcional)
└── screenshots/        # Capturas de pantalla de la app (opcional)
```

## Branding

- Guías de uso de marca
