# Financial Copilot - Frontend

## 🚀 Comenzar

### Instalación

```bash
npm install
```

### Desarrollo

```bash
npm run dev
```

Abrir [http://localhost:3000](http://localhost:3000)

### Build

```bash
npm run build
npm start
```

## 📁 Estructura

```
src/
├── app/              # Pages (Next.js App Router)
│   ├── page.tsx      # Login/Register
│   └── dashboard/    # Dashboard
├── components/       # Componentes reutilizables
└── lib/             # Utilidades (API client)
```

## 🎨 Stack

- Next.js 14
- React 18
- TypeScript
- TailwindCSS
- Axios
